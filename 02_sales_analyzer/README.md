# 02 - Pandas Basics

This project introduces basic data analysis using Pandas.

The project works with sales data and explores loading CSV files, inspecting data, creating calculated columns and performing aggregations.

## Analysis

The dataset was analyzed to find:

- Total revenue
- Revenue by product
- Best performing product
- Best performing category
- Average revenue per sale

## Conclusion

Pandas provides powerful tools for organizing and analyzing structured data. This project was my first step into practical data analysis.

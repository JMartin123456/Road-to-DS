# 05 - NumPy Statistics

This project introduces numerical analysis using NumPy.

The employee salary data was analyzed using NumPy arrays to calculate statistical values and detect extreme salary values.

## Analysis

The project calculated:

- Minimum salary
- Maximum salary
- Mean
- Median
- Standard deviation
- Variance
- Number of records

Outliers were detected using the IQR method and the effect of extreme values on salary statistics was compared.

## Conclusion

NumPy provides useful tools for numerical analysis and is an important foundation for data analysis and machine learning.
